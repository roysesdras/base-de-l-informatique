Quelles compétences allez-vous acquérir ?

1. Les bases de l’informatique

Les bases de l’informatique, comme le fonctionnement d’un ordinateur, les périphériques, les systèmes d’exploitation, les navigateurs web, les logiciels, etc.

2. La programmation

La programmation, qui consiste à créer des instructions pour faire exécuter des tâches à un ordinateur. Il existe de nombreux langages de programmation, comme le C, le Java, le Python, le PHP, etc.

3. Les réseaux informatiques

Les réseaux informatiques, qui permettent de connecter des ordinateurs entre eux, localement ou à distance, pour échanger des données. Vous apprendrez les concepts de base des réseaux, comme les adresses IP, les protocoles, les routeurs, les serveurs, etc.
